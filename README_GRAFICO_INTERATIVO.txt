INVCRIPTO IA - Gráfico Interativo v1.0.5

Correções incluídas:
- Timeframes 1m, 5m, 15m, 1h, 4h e 1D agora alteram os candles reais.
- O gráfico pode ser movimentado arrastando para esquerda/direita.
- Scroll do mouse aproxima/afasta o gráfico.
- Botões no gráfico: voltar, zoom +, zoom -, LIVE e avançar.
- Duplo clique no gráfico volta para o candle atual.
- Mantido gráfico nativo sem lightweight-charts para evitar erro de deploy no Netlify.

Após subir no GitHub, no Netlify use Clear cache and deploy site.
