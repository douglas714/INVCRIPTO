# Layout Premium INVCRIPTO IA

Esta versão aplica o visual verde/dourado inspirado na identidade INVCRIPTO:

- Logo e favicon em `public/invcripto-logo.png`, `public/favicon.png` e `public/favicon.ico`.
- Dashboard com cards superiores, gráfico real, suporte/resistência, Trading Control, Radar IA e System Performance.
- Paleta principal:
  - Verde escuro: `#061B12`
  - Verde neon: `#16C957`
  - Dourado: `#D8B24A`
  - Dourado escuro: `#8A6A22`
  - Fundo: `#050806`
  - Texto claro: `#F5EBC8`

O mockup de referência está em `docs/mockup_layout_premium_invcripto.png`.
